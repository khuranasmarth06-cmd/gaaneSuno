# Gaane Sunno - Music Player Website
Gaane Sunno is a stylish and interactive music player website built using HTML, CSS, and JavaScript. It allows users to browse playlists, search songs using an API, play music, and control playback with a smooth UI.
---
## Live Features
* Play / Pause songs
* Next / Previous track
* Interactive progress bar
* Multiple playlists (Honey Singh, Dhurander)
* Search songs using Deezer API
* Navigation between pages
* Modern glassmorphism UI
* Fast and lightweight
---
## API Integration
### Deezer API
* Fetch songs dynamically based on user search
* Display song title, artist, album cover
* Play 30-second preview audio
* Real-time search results
---
## Pages Overview
### Landing Page (index.html)

* Welcome screen
* Navigate to songs or info page
### Playlist Page (index2.html)
* Select artist playlist
* View songs dynamically
* Music player controls
### Search Page (index3.html)
* Search songs using Deezer API
* Display results in card layout
* Play song previews
---
## Technologies Used
* HTML5 (Structure)
* CSS3 (Styling, animations, glassmorphism)
* JavaScript (Vanilla, functionality and API handling)
* Font Awesome (Icons)
* Deezer API (Music data and preview streaming)
---
## How to Run
1. Download or clone the repository
2. Open index.html in your browser
3. Navigate through pages
4. Use search to fetch songs via API
---
## How It Works
* Songs are stored in arrays inside script2.js
* Deezer API is used in script3.js for searching songs
* Fetch API retrieves song data dynamically
* Audio preview is played using the audio element
* Progress bar updates in real time
* Navigation is handled using window.location.href
