# Gaane Sunno - Music Player Website
Gaane Sunno is a stylish and interactive music player website built using HTML, CSS, and JavaScript. It allows users to browse playlists, play songs, and control music with a smooth UI.
---
## Live Features
* Play / Pause songs
* Next / Previous track
* Interactive progress bar
* Multiple playlists (Honey Singh, Dhurander)
* Navigation between pages
* Modern glassmorphism UI
* Fast and lightweight
---
## Pages Overview
### Landing Page (index.html)
* Welcome screen
* Navigate to songs or info page
### Playlist Page (index2.html)
* Select artist playlist
* View songs dynamically
* Music player controls
### Coming Soon Page (index3.html)
* Placeholder for future features

## How to Run
1. Download or clone the repository
2. Open index.html in your browser
3. Use the buttons to navigate between pages
---
## How It Works
* Songs are stored in arrays inside script2.js
* Clicking a playlist dynamically displays songs
* Audio is handled using the JavaScript Audio() object
* Progress bar updates in real time
* Navigation is handled using window.location.href
